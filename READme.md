# Modern TTS Studio 🎙️

A professional Text-to-Speech desktop application featuring a sleek Dark Mode UI and full audio export capabilities.

## Features
- **Modern UI**: Built with `CustomTkinter` for a native, high-end feel.
- **Dark Mode**: Reduces eye strain with a dark-themed interface.
- **Voice Customization**: Choose from system-installed voices.
- **Precision Control**: Fine-tune speech rate and volume with smooth sliders.
- **Export to WAV**: Save your narrations for videos or presentations.
- **Multi-threaded**: No UI lag while processing speech.
